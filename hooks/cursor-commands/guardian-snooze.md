# Guardian: snooze prompt gates (15 minutes)

**Invoking in Cursor:** In the chat composer, type **`/guardian-snooze`** and press Enter — Cursor inserts this page (commands come from `~/.cursor/commands/guardian-snooze.md` after `hooks/install-hooks.sh`).

Run in **Terminal** (or Cursor’s integrated terminal):

```bash
~/.guardian/guardian snooze 15
```

**macOS:** double-click `~/.guardian/Guardian-Snooze-15m.command` in Finder, or run:

```bash
open "$HOME/.guardian/Guardian-Snooze-15m.command"
```

Then submit your prompt again in Cursor.
